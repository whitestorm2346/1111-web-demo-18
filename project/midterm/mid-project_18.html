Program Projects

Sorting Simulator

I used the SFML library to implement this project. The simulator can sort 50 ~ 5000 blocks using different sorting algorithms. The way that algorithms work will be present in the window. Then we can compare the time complexity of these algorithms.

c++

Snake

Snake is a classic game that is easy to implement using some basic queue knowledge. For the beginner, you can learn how to set the position of the cursor, and learn where to print out the snake's body part. So it's a useful mini-project.

c++

Auto-class-choosing

This project is created because choosing a class is hard at TKU. Speed is the major part of choosing the class. So I figure out how to log in to the class-choosing website and choose the classes I wanted automatically.
